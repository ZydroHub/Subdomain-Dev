What is NOVA?

Project Overview

NOVA is a desktop-based, voice-first AI assistant running entirely on a Raspberry Pi 5 8 GB. It delivers a fast, responsive, and visually cinematic experience — fully offline and capable of controlling personal devices without any cloud dependency.

Voice-Controlled

Streaming TTS with interruptible speech and wake-word detection, all running locally.

Fully Offline

Modular offline-first design. No cloud, no subscription — your data stays on device.

Cinematic UI

Animated display with dynamic text streaming and subtle idle animations — Jarvis-grade experience.

Modular Design

Expandable with local hardware: microcontrollers, audio devices, and optional sensors.

Multi-Category Knowledge

News, calendar, weather, music, smart-home — all accessible via natural voice commands.

Raspberry Pi 5 Native

Optimized for Raspberry Pi 5 8 GB — efficient LLM, STT, and TTS models purpose-built for the hardware.

Core Capabilities

Feature Highlights

Streaming TTS & Interruptible Speech

Real-time text-to-speech pipeline that can be cut mid-sentence via voice command, just like a real conversation.

Offline LLM Inference

Runs quantised language models directly on-device — no API calls, no latency spikes, full privacy.

Smart Scheduling & Reminders

Built-in calendar layer with background fetcher for news updates, reminders, and data refresh tasks.

Home & Device Control

Control personal devices and smart-home hardware through natural-language commands via the Home category layer.

Music & Media Playback

Integrated music category layer — ask NOVA to play, pause, skip, or queue tracks entirely by voice.

Weather & News

Background scheduler fetches weather data and news digests, ready to brief you on demand — no internet required at query time.

Under the Hood

Core Architecture

Conversation Manager

Handles context, intent routing, response streaming, and mid-sentence interruptions.

Category Layer

Manages knowledge sources: News, Calendar, Weather, Music, and Home Screen.

Scheduler / Fetcher

Background task engine for news updates, data refresh, and timed reminders.

UI Manager

Controls the screen: animations, text streaming, and real-time status indicators.

Audio Manager

Handles input capture, STT, TTS, audio streaming pipelines, and interruption signals.

Models Layer

LLM, STT, TTS, and function-calling models — all offline-optimized for Raspberry Pi 5.
